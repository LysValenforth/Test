TOPIC 01
Normal Curve
Z-SCORES · AREAS · PERCENTILES · APPLICATIONS
CONCEPT

What is a Z-score?

Measures how far a score is from the mean in standard deviations. Z=+1 means 1 SD above; Z=−2 means 2 SDs below.

FORMULA

Z = (x − x̄) / s

x = observed score, x̄ = mean, s = standard deviation. Positive Z → above mean. Negative Z → below mean.

READING AREAS

Using the Z-Table

Table gives area from mean (Z=0) to your Z. Full curve = 1.0; each half = 0.5. Add or subtract areas as needed.

REVERSE LOOKUP

Finding Raw Score

Rearrange formula: x = Z·s + x̄. Used when you know the percentile cutoff and need the actual score.

📘 Worked Example 1 — Computing Z-Scores

Given: mean = 65, SD = 6. Find Z-scores for scores 68, 59, 77, 47.

Score (x)FormulaZMeaning
68(68−65)/6+0.500.5 SD above mean
59(59−65)/6−1.001 SD below mean
77(77−65)/6+2.002 SDs above mean
47(47−65)/6−3.003 SDs below mean
📘 Worked Example 2 — Area Under the Curve

Scores normally distributed, mean = 30, SD = 5.

a

≥ 30: Z = 0. Right half of curve = 50%

b

≥ 37: Z = 1.4. Area from table = 0.4192. Right tail = 0.5 − 0.4192 = 8.08%

c

28 to 34: Z₁=−0.4 (0.1554) + Z₂=0.8 (0.2881) = 44.35%

📘 Worked Example 3 — Finding the Cutoff Score

Mean = 200, SD = 20. Find the lowest value in the upper 10%.

1

Upper 10% is at the right tail. Area from mean to boundary = 0.5 − 0.10 = 0.40

2

Look up 0.40 in Z-table → closest 0.3997 → Z ≈ 1.28

3

x = Z·s + x̄ = (1.28)(20) + 200 = 225.6 → x = 226

TOPIC 02
Hypothesis Testing
7-STEP PROCESS · Z-TEST · T-TEST · ERRORS
H₀ — NULL

No Significant Difference

Uses =, ≥, or ≤. Default claim — rejected only when evidence strongly contradicts it.

H₁ — ALTERNATIVE

Significant Difference

Uses <, >, or ≠. Accepted when H₀ is rejected. Its direction determines the tail type.

TYPE I ERROR — α

False Rejection

Rejecting H₀ when it's actually true. Controlled by setting α = 0.01 or 0.05.

TYPE II ERROR — β

False Acceptance

Accepting H₀ when it's actually false. Related to test power = 1 − β.

📋 The 7 Steps

1

State H₀ and H₁ — write clear null and alternative hypotheses.

2

Set α — level of significance: 0.01 (1%) or 0.05 (5%).

3

Identify test type — H₁ with < or > → one-tailed. H₁ with ≠ → two-tailed.

4

Find tabular value — from Z-table or T-table based on α, tail type, and df.

5

Compute test statistic — apply Z or T formula with given data.

6

Decision — |computed| > tabular → Reject H₀. Otherwise → Accept H₀.

7

Conclusion — interpret the result in real-world context; don't just say "H₀ rejected."

Z-TEST

Large Samples · σ Known

Use when population σ is known and n ≥ 30.

Z = (x̄ − μ)√n / σ
T-TEST

Small / Unknown σ

Use when σ is unknown — even if n > 30.

t = (x̄ − μ)√(n−1) / s

📐 Degrees of Freedom (df)

Two formulas — pick based on your test type:

Test TypeFormulaExample
One-sample t-testdf = n − 1n=20 → df=19
Two-sample independent t-testdf = n₁ + n₂ − 2n₁=25, n₂=30 → df=53
Paired / dependent t-testdf = n − 1n=10 pairs → df=9
📘 Example: Z-test (One Sample) — Insect Wings

50 insects: x̄=17.4mm. Known species: μ=15.4, σ=2.3. α=0.05.

StepValue / Action
H₀μ = 15.4 (same species)
H₁μ ≠ 15.4 → two-tailed
Z tabular±1.96
Z computed(17.4−15.4)√50 / 2.3 = 6.15
Decision6.15 > 1.96 → Reject H₀
ConclusionInsects are of a different species.
📘 Example: T-test (One Sample) — Perfume Alcohol

20 bottles: x̄=29.7%, s=4.8%. Claim: ≤25%. α=0.05.

StepValue / Action
H₀μ ≤ 25
H₁μ > 25 → one-tailed (right)
dfn−1 = 19 → t tabular = 1.729
t computed(29.7−25)√19 / 4.8 = 4.27
Decision4.27 > 1.729 → Reject H₀
ConclusionPerfume has significantly more than 25% alcohol.
📘 Example: T-test (Two Independent) — Mendiola vs Recto

Mendiola: n=25, x̄=245, s=103. Recto: n=30, x̄=438, s=79. α=0.05.

StepValue / Action
H₁x̄₁ < x̄₂ → one-tailed (left)
df25 + 30 − 2 = 53 → t tabular = −1.674
t computed(245−438) / √(103²/25 + 79²/30) = −23.46
Decision|−23.46| > 1.674 → Reject H₀
ConclusionMean indices in Mendiola are less than in Recto.

STANDARD SCORE
Z-Score

x = observed value
x̄ = mean

Z-TEST · ONE SAMPLE
Large Sample vs Population
Z = (x̄ − μ)√n / σ
x̄ = sample mean
μ = population mean
σ = population SD · n = size
Z-TEST · TWO SAMPLES
Comparing Two Large Means
Z = (x̄₁−x̄₂) / σ√(1/n₁+1/n₂)
x̄₁, x̄₂ = sample means
σ = population SD
n₁, n₂ = sample sizes
T-TEST · ONE SAMPLE
Small Sample / Unknown σ
t = (x̄ − μ)√(n−1) / s
x̄ = sample mean
μ = population mean
s = sample SD · df = n−1
T-TEST · INDEPENDENT
Two Sample Means (Indep.)
t = (x̄₁−x̄₂) / √(s₁²/n₁+s₂²/n₂)
s₁, s₂ = sample SDs
n₁, n₂ = sizes
df = n₁ + n₂ − 2
T-TEST · DEPENDENT/PAIRED
Two Sample Means (Dep.)
t = x̄_D / (s_D / √n)
x̄_D = mean of differences
s_D = SD of differences
df = n−1 (n = pairs)
DEPENDENT SD
s_D Formula
s_D = √[Σ(x_D−x̄_D)²/(n−1)]
x_D = each pair difference
x̄_D = mean of differences
